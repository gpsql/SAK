# Laboratorium 01: Scratch – Logika Animacji i Symulacja Ruchu

## Opis Projektu
Projekt zrealizowany w środowisku Scratch. Zawiera dwa gówne zadania:
1. **Zadanie A: Symulacja Rzutu Ukośnego (Fizyka)** - obiekt porusza się zgodnie z zasadami dynamiki (grawitacja, prędkość, suwaki do parametryzacji).
2. **Zadanie B: Animacja Układu Słonecznego (Kinematyka)** - hierarchiczny model ruchu planetarnego wykorzystujący trygonometrię.
